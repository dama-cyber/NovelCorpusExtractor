# 语料库示例目录

本目录用于存储提取的Frankentexts语料片段。

## 文件说明

- `06_玄幻预料库.txt`: 玄幻类小说语料
- `07_都市预料库.txt`: 都市类小说语料
- `08_言情预料库.txt`: 言情类小说语料
- `09_悬疑预料库.txt`: 悬疑类小说语料
- `10_历史预料库.txt`: 历史类小说语料
- `11_科幻预料库.txt`: 科幻类小说语料
- `12_武侠预料库.txt`: 武侠类小说语料
- `13_对话预料库.txt`: 对话语料

## 语料格式

每个语料片段包含以下信息：

```
================================================================================
片段ID: FRAG_20251120105730_1234
类型: 战斗场景
提取时间: 2025-11-20T10:57:30
元数据: {
  "novel_type": "玄幻",
  "chunk_id": "ch1_c5",
  "tags": ["战斗", "高潮", "转折"]
}

原文:
[原文内容...]

模板:
[模板化后的内容，专有名词替换为占位符...]
================================================================================
```

## 使用说明

系统会自动将提取的高价值片段保存到对应类型的语料库文件中。这些片段可用于后续的文本生成和拼接。
